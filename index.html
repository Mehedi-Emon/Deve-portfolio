Vanilla JavaScript

Vanilla JavaScript refers to plain JavaScript without using any libraries or frameworks like React, jQuery, or Angular. It's the core language of the web, used to create dynamic and interactive elements on websites.

React

React is a JavaScript library developed by Facebook for building fast and interactive user interfaces, especially single-page applications. It uses a component-based architecture and a virtual DOM to efficiently update and render UI elements.

Node. JS

Node.js is a runtime environment that allows you to run JavaScript on the server side instead of just in the browser. It's built on Chrome's V8 engine and is great for building fast, scalable network applications like APIs and real-time services.
